Introducing DICE
================

DICE - originally an acronym for *Dillon's Integrated C Environment* - is a C compiler and suite of
related tools. It targets the Commodore Amiga series of computers, and can also build for
68000-based embedded systems.

DICE has a long history, starting off as a ShareWare project in the 1980s. Most recently, it has
been open-sourced by the original author under the BSD License. This fork is now maintained by
open-source contributors.
